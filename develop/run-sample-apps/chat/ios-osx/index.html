运行聊天样例 - iOS

前提条件

  • Build samples
  • 在两个 iOS 设备上安装 Basic Client 和 Basic Service sample 应用程序。
  • 两个 iOS 设备必须连接到同一个 Wi-Fi 网络中。

运行 AllJoyn™ 聊天

在两个设备上运行 AllJoyn 聊天应用程序。

现在两个 iOS 设备上都应该运行着 AllJoyn 聊天应用程序。初始界面应是如下图所示：

Figure: 聊天初始化界面

使用会话的聊天

  1. 确保两个设备上的 Use a session 选项都被放置在 ON 状态上。

  2. 在一方设备上，选中 Host 按钮，然后按下 Start. 在另一设备上，选中 Join 按钮，然后按下 Start.两方的应用程序界面都应如下图 所示：

Figure: 初始化消息界面

  1. 在一方设备上，键入一条消息并按下 Send. 此消息出现在发送端的设备上，同时也出现在另一方设备上。截图如下：

Figure: 设备1发送 "hello"

Figure: 设备2收到设备1发送的 "hello"

Figure: 设备2回复 "hi"

Figure: 设备1收到设备2回复的 "hi"

使用非会话信号的聊天

  1. 确保两个设备上的 Use a session 选项都被放置在 OFF 状态上。

Figure: 初始化聊天界面

  1. 在两设备上按下 Start. 两方的应用程序界面都应如下图所示：

Figure: 初始消息界面

  1. 在一方设备上，键入一条消息并按下 Send. 此消息出现在发送端的设备上，同时也出现在另一方设备上。截图如下：

Figure: 设备1键入 "hi"

Figure: 设备1 收到设备2回复的 "hey"
